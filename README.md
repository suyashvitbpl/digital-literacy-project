The objective of this project is to develop a deep understanding of Digital Literacy and establish a foundational Professional Identity. In an era where cyber threats like UPI fraud and phishing are becoming increasingly, it is no longer enough to just write code, one must also navigate the digital world with care and security habits.
This project document outlines a series of practical tasks designed to bridge the gap between technical skills and digital awareness. These include:
Security Awareness: Analyzing real-world case studies of online payment fraud and creating educational resources like a Google Forms quiz and Canva infographics.
Professional Begining: Setting up for platforms like LinkedIn, GitHub, and Kaggle to document four years of academic growth.
Technical Proficiency: Demonstrating hands-on coding practice through HackerRank Python challenges.
Ultimately, this project serves as a reflection of my commitment to becoming a responsible digital citizen and a technically sound engineer, prepared for the evolving demands of the global tech industry.
